GM Voice Studio — Image Assets
================================

Drop your image files into the correct subfolder.
Any format works: .jpg  .png  .webp  .svg

portraits/
  aethelred.jpg   — Paladin portrait
  lira.jpg        — Rogue portrait
  torin.jpg       — Wizard portrait
  zephyr.jpg      — Druid portrait
  mira.jpg        — Cleric portrait

tools/
  dice.jpg        — Roll Dice card
  bestiary.jpg    — Monster Bestiary card
  bluff.jpg       — Spell Ref / Roll Bluff card
  npc.jpg         — Gen NPC card
  loot.jpg        — Loot Table card
  damage.jpg      — Apply Damage card

nav/              — (optional) footer nav icon overrides
map/              — (optional) map pin / castle images

Files are served at: /static/img/<subfolder>/<filename>
No server restart required — just drop the file and refresh the browser.
